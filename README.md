## pins

### relais.

- Relay 1: 28
- Relay 2: 26

### bumper:

- Start: 53
- Softdrinks: 51
- Alcohol 1: 49
- Alcohol 2: 47
- Alcohol 3: 45
- End: 43 (Also alcohol 4)

pulse = 52;
